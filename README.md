# Checkout Journal

A lightweight CLI journaling app for 5-minute evening reflection.

## Installation

```bash
npm install
npm link
```

## Usage

```bash
checkout           # Start new entry
checkout list      # View entries
checkout import    # Import markdown file
checkout validate  # Check file integrity
checkout config    # Show/edit config
```
